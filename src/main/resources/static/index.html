Pell Sequence

Calcular la secuencia de Pell hasta un índice dado: pellseq(n) retorna un JSON con los términos desde P(0) hasta P(n). (Recibe solo enteros no negativos). Definición P(0) = 0 P(1) = 1 P(n) = 2P(n-1) + P(n-2)

Ejemplo de llamado (EC2) https://amazonxxx.x.xxx.x.xxx:{port}/pellseq?value=13

Salida (JSON) { "operation": "Secuencia de Pell", "input": 13, "output": "0, 1, 2, 5, 12, 29, 70, 169, 408, 985, 2378, 5741, 13860, 33461" }
